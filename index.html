kowm-opts

kowm-opts ( /kōm-äpts/ ) currently has at least six additional modular libraries planned. These libraries - along with any others we think might be needed - will provide additional services that, while not absolutely necessary, would probably be appreciated by the end user. These libraries will most likely be written in Ruby; however additional bindings for Python and Lua are currently on the RoadMap.
